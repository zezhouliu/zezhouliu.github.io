CS171: Visualizations


Course Description

CS171 is a great course that teaches how to harness data and transform it into a visually appealing and informative visualization. We learned topics including good design practices for visualization, tools for visualization of data, and methods and techniques to increase the understanding of complex data. We applied these techniques by implementing interactive web visualizations using d3.js.


Goals

1. How do we use visualizations to communicate trends and help with understanding increasingly large and complex data sets?
2. What makes a visualization a good visualization?
